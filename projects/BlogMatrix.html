BlogMatrix my very simple blog engine

I've always wanted to create my own blog engine purely for the heck of it and the geek factor. Over the years I wrote various little attempts with varying success but in the end I always lost steam because I was trying to replicate WordPress whilst learning the 'latest and greatest programming language'.

In the end I sat down and tried to figure out what was the bare minimum blog engine I could come up with and what were the goals. I eventually settled on the following:

  1. Can be hosted for free
  2. Uses git for deployment
  3. No database
  4. Scrap bloated development frameworks ie. ASP.NET MVC, Rails
  5. Has mechanism for layouts and includes
  6. Responsive design

The solution I came up with was BlogMatrix which you can view on GitHub. How simple is it you ask? Well here are the solutions I came across for each of the goals above.

1. Can be hosted for free
2. Uses git for deployment
3. No database

Solution: GitHub Pages. By now I'm hoping you've heard of GitHub and are using it for hosting your projects. What some people don't realise is that you can also use it for simple html hosting. Instructions on how to set this up are here.

Obviously with it being hosted by GitHub they would allow you to deploy using git which they do.

Likewise the whole site is pure html so there is no database.

4. Scrap bloated development frameworks ie. ASP.NET MVC, Rails
5. Has mechanism for layouts and includes

Solution: WebMatrix. WebMatrix is a very simple development environment which is simplar in concept to PHP and classic ASP. It provides some of the power of ASP.NET MVC (razor view engine, layouts etc) without the added bloat. Here's an excellent tutorial series by Rob Conery at TekPub.

6. Responsive design

Solution: Twitter Boostrap with Metro-Bootstrap Theme. This was a no brainer. Twitter Bootstrap is an excellent HTML/CSS/jQuery framework with all the bells in whistles. It has built in support for creating responsive design websites. I used the Metro-Bootstrap theme because I liked the look of it and flat design is 'hot right now'.
